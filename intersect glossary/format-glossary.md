Basic Format Standard for Glossary
# Basic Format

  Glossary
  -List (Req)
  
	- Entry
		- **Term** (Req)
		- *Definition* (Req) 
		- *Second Definition* (Opt)
		- Example (Opt)
		- Type  (Opt)
			- Broad 
		    - Narrow
		    - Related
		- Category (Opt)
	    - Style (Opt)
	    - Entered By (Req)
	    - Date (Req)
	    - Source (Opt) 
	    - Verified By (Req)

New File Example:
## ListName 
<dl>
  <dt>First Term</dt>
  <dd>This is the definition of the first term.</dd>
  <dd>Entered By: </dd>  
  <dd>Date: </dd>
  <dd>Verified By: </dd>
  <dt>Second Term: </dt>
  <dd>This is one definition of the second term. </dd>
  <dd>This is another definition of the second term.</dd>
</dl>
